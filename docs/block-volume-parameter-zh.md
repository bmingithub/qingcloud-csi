# 存储类型参数说明

 本节列出参数仅供参考，用户设置存储类型中的参数应以实际情况为准。

 ## 硬盘类型与 type 参数对应关系

 |硬盘|Volume|type 值|
|:---:|:----:|:----:|
|性能型| High Performance|0|
|容量型| High Capacity|2|
|超高性能型|Super High Performance|3|
|NeonSAN| NeonSAN|5|
|基础型| Basic|100|
|SSD 企业型| SSD Enterprise|200|
